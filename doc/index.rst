Puckfetcher
===========

Puckfetcher is a simple command-line podcatcher.

Running it:

> puckfetcher menu

	puckfetcher 1.3.0 started!
	Choose a command
	[1] Exit application.
	[2] Update all subscriptions. Will also download sub queues.
	[3] List current subscriptions and their status.
	[4] Provide details on one subscription's entries and queue status.
	[5] Add to a sub's download queue. Items will be skipped if already in queue, or invalid.
	[6] Mark a subscription entry as downloaded.
	[7] Mark a subscription entry as not downloaded. Will not queue for download.
	[8] Download a subscription's full queue. Files with the same name as a to-be-downloaded entry will be overridden.
	[9] Summarize subscription entries downloaded in this session.
	[10] Summarize recent entries downloaded for a specific sub.
	[11] Reload configuration file.

Features
--------

- Straightforward configuration.
- Downloads any well-formatted podcast.
- Built-in rate-limiting.
- Cleans up MP3 filenames and tags.

Installation
------------

Puckfetcher is available on pypi, and requires Python 3.6+.

    pip3 install puckfetcher
